# **App Name**: CommunityPulse

## Core Features:

- User Authentication: Secure login and signup system with automatic 'test' account creation.
- Dashboard with KPIs and Charts: Display key statistics (total issues, resolved tickets) and visualize trends using charts.
- AI-Powered Intelligence Hub: Use AI with anomaly detection tools to give users predictive alerts.
- Issue Reporting with AI Category Suggestion: Allow users to report issues and automatically suggest issue categories using AI.
- Smart Map Page: Visually represent reported issues on a map with details and images.
- Live Community Feed: Real-time feed displaying latest reported issues, reporter names, and upvote counts.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to convey trust and stability.
- Background color: Light Indigo (#E8EAF6), a desaturated tint of the primary for a calm and uncluttered backdrop.
- Accent color: Violet (#7952B3) to draw attention to key interactive elements and AI features.
- Body and headline font: 'Inter' (sans-serif) for a modern, machined, objective, neutral look. 
- Use clear, consistent icons from a library like Material Design to represent issue categories.
- Employ a clean, grid-based layout for dashboards and data visualization, ensuring information is easily accessible.
- Use subtle animations to enhance user experience (e.g., loading indicators, transition effects, live feed updates).
